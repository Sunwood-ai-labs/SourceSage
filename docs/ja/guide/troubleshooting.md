# トラブルシューティング

## 要約に `.git` の中身が出る

通常は `.git/` はデフォルトで除外されます。もし Git 内部ファイルが出る場合は、渡した ignore ファイルがデフォルト挙動を上書きしていないか確認してください。

## `uvx` の古いキャッシュが使われる

```bash
uv cache clean
uv cache prune
```

## 独自の ignore ルールを使いたい

対象リポジトリの `.SourceSageignore` を編集してから、次を再実行します。

```bash
uv run sage --repo .
```

## diff レポートが生成されない

`--diff` は Git タグが 2 つ以上必要です。タグが足りない場合は、コマンド全体を失敗させずに diff レポートだけをスキップします。
