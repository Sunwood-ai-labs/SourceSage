# Security Policy

## Supported Versions

Security fixes are prioritized for:

- the latest published PyPI release
- the current `main` branch

## Reporting a Vulnerability

Please do not open a public GitHub issue for security reports.

Instead, contact the maintainers at `sunwood.ai.labs@gmail.com` with:

- a clear description of the issue
- steps to reproduce it
- the affected version or commit
- any proposed mitigation or workaround, if available

We will acknowledge reports as quickly as possible and coordinate a fix before public disclosure when needed.
